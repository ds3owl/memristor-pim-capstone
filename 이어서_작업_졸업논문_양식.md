## 3. 한글 COM 편집 패턴 (PowerShell, 검증됨)
```powershell
Get-Process | ? { $_.Name -match 'Hwp|Hncom' } | Stop-Process -Force   # 잔존 프로세스 먼저 종료 (안 하면 SaveAs=False)
$hwp = New-Object -ComObject "HWPFrame.HwpObject"
try { $hwp.RegisterModule("FilePathCheckDLL","FilePathCheckerModule") } catch {}
$hwp.Open($f, "HWPX", "")
# 찾기·바꾸기:
$fr=$hwp.HParameterSet.HFindReplace; $hwp.HAction.GetDefault("AllReplace",$fr.HSet)
$fr.FindString="X"; $fr.ReplaceString="Y"; $fr.IgnoreMessage=1; $fr.Direction=0; $fr.ReplaceMode=1
$hwp.HAction.Execute("AllReplace",$fr.HSet)   # 반환값 False여도 실제 적용될 수 있음 → 결과 확인 필수
# 쪽 나눔 삽입: MoveDocBegin → RepeatFind로 위치 → MoveParaEnd → Run("BreakPage")
$hwp.SaveAs($f,"HWPX",""); $hwp.SaveAs($pdf,"PDF","")   # PDF로 검증
$hwp.Quit()
```
- 글자크기 변경은 XML 수술도 가능: hwpx 압축풀어 `Contents/header.xml`의 charPr `height` 변경(1050=10.5pt, 1100=11pt), mimetype 무압축 첫 항목으로 재패키징.

# 3개 외부 LLM 신규 지적 통합 — paper.html v3(옵션 D 적용 후) 대상

## 종합 평가
- LLM-A: 82~85점, "학부 캡스톤 안정권에 가까움". VTEAM 표현 톤다운 + 6개 수정
- LLM-B: 미해결 2건 중심, 우선순위 1·2가 핵심
- LLM-C: 심사위원 톤으로 5개 약점 + 가독성 1개

## 통합 신규 지적 (15개, 중복 합침)

### CRITICAL — 3 LLM 공통 약점

**M9. Nanowire 5% fault rate 형평성 문제 (3 LLM 모두 강조)**
- NW만 5% fault rate, 타 3종은 0.3~0.5% → 10배 차이가 confound
- LLM-B: "감도 분석을 위한 의도적 보수 가정(sensitivity bound)" framing 변경 + NW fault=0.5% 1회 추가 권고
- LLM-C: "결함률 비이상성 주입 조건을 10배 다르게 설정하고 수평 비교하는 것은 논리적 어폐"
- → **두 옵션**: (a) framing 변경(텍스트만), (b) NW fault=0.5% sensitivity 실험 추가

**M1+M2. "VTEAM 시뮬레이션 환경" / "사용한 핵심 방정식" 표현 (LLM-A, LLM-C 공통)**
- 실제 방법론은 정적 conductance mapping + 양자화 + variation/fault
- VTEAM 동역학(v_on, k, α, w(t) 적분)은 사용 안 함
- → 표현 톤다운: "VTEAM 문헌 파라미터를 참고한 정적 conductance mapping 환경" / "참조한 기본 방정식"

### HIGH — 사실 오류 또는 검증 불가 주장

**M8. "예비 실험에서 차이 미미" 주장의 데이터 부재 (LLM-B 우선순위 1)**
- III.D-3 σ 평균 처리 근거로 "예비 실험" 언급했으나 수치 없음
- → 표현 약화: "구현 단순화를 위한 근사이며, 분리 적용 결과 변동 가능성은 한계 절에 명시"

**M3. "스위칭 임계 전압 등" vs Table 2 (LLM-A)**
- III.C에서 "R_on, R_off, 스위칭 임계 전압 등"이라 했으나 Table 2엔 v_on/v_off 없음
- → 단순 수정: "R_on, R_off 등"으로 변경

**M5. "실제 PIM 칩에서 빈번히 보고되는 catastrophic" 과함 (LLM-A)**
- → "실제 PIM 칩에서 발생할 수 있는 심각한 성능 저하"로 톤다운

**M12. Pt 0.7578±0.2759를 "유지"로 묶음 (LLM-C)**
- std 0.27인데 "분류 성능을 어느 정도 유지"는 과대
- → 초록 표현 수정 ("유지" → "변동성이 큰 가운데 평균 성능을 보존")

### MEDIUM — 통계·표현 보강

**M10. "약 11%" 정밀도 과장 (LLM-B 우선순위 3)**
- n=9에서 1회 binomial 95% CI 약 [0.3%, 48%]
- → V장에 "표본 한계상 점추정치, 정확한 발생률은 더 많은 반복 필요" 한 줄

**M11. NW 결과 해석과 confounding 한계 정합성 (LLM-B 우선순위 4)**
- ideal 0.38이 "NW 본질 한계"인지 "낮은 저항비→4-state 강제 할당"인지 분리 안 됨
- → IV에 한 줄: "이상 조건 붕괴는 4-state 양자화 효과와 분리 평가되지 않았으며, ablation은 향후 과제"

**M13. N=9 통계 신뢰성 (LLM-C)**
- 30~50회 권장 또는 정성적 표현
- → M9와 함께 sensitivity 보강 + M10 톤다운으로 일부 해결

**M14. F1 변화 % vs %p 표기 (LLM-C)**
- "−22.88%"가 비율 감소인지 절대 차인지 모호
- → "F1 Change" 열을 절대값(예: −0.2248) 또는 %p로 변경

**M4. "비가역적으로 변화" (LLM-A)**
- 비가역은 부정확 (SET/RESET으로 재변경 가능)
- → "인가 전압 이력에 따라 저항 상태가 변화하고, 비휘발성 2단자 소자"

**M6. "통계적 신뢰성 확보" → "seed 의존성 확인" (LLM-A)**
- 3 seed가 통계 신뢰 확보 표현엔 부족
- → 표현 약화

**M7. R(w) 식 정규화 변수 명시 (LLM-A)**
- "여기서 w는 0~1로 정규화된 내부 상태 변수이다" 한 줄 추가

### LOW — 가독성/표기

**M15. 영문 용어 한글 통일 (LLM-C)**
- catastrophic degradation, device-to-device variation 등 → 한글 병기 후 일관 사용

## 4. 연구 핵심 요약 (paper.html 안 봐도 답할 수 있게)

### 방법론
- **데이터셋**: MixedWM38 (Wang 2020 IEEE TSM), 37,015장 (normal 1,000 제외), 8 클래스 multi-label
- **CNN**: 3 Conv + FC(128) + Dropout(0.3) + FC(8), 319,592 파라미터
- **4 소재 비교**: Pt/HfOx/Ti (25×), Ferroelectric (333×), Metallic Nanowire (**1.97×, 4 state**), TaOx/Ta (146×)
- **3 비이상성 동시 적용**: 양자화 + variation + stuck-at fault
- **평가**: Ideal/Realistic 두 조건, 9 Monte Carlo (3 baseline seed × 3 noise sample), Macro F1 + Subset Acc

### 핵심 결과
| Material | Ideal F1 | Realistic F1 |
|---|---|---|
| Pt/HfOx/Ti | 0.9826 | 0.7578 ± 0.2759 |
| Ferroelectric | 0.9805 | 0.8570 ± 0.1632 |
| **Metallic Nanowire** | **0.3847 (망함!)** | 0.3288 ± 0.1286 |
| TaOx/Ta | 0.9770 | 0.8726 ± 0.1364 |

- FP32 baseline: 0.9812 ± 0.0032
- **고저항비 3 소재 각각 9회 중 1회 worst-case 붕괴** (F1 < 0.60)
- **NW는 ideal에서도 분류 실패** (4-state 양자화의 본질적 한계)
- **Per-class F1**: Near-Full(0.4%), Scratch(2.3%) minority class가 가장 약함

### 결론 (3 가이드라인)
1. R_off/R_on 25배 이상 확보
2. 평균 성능 + worst-case 위험 함께 평가
3. fault tolerance 메커니즘 (write-verify, ECC, redundancy) 병행 필요성 제안

## 5. 알려진 한계 (의도적으로 안 건드린 부분, V.D에 명시됨)

- 단일 데이터셋 (MixedWM38만), 경량 CNN(32만 파라미터)
- **저항비 sweep 미수행** (4 소재 이산 비교만)
- **회로 수준 비이상성 미반영** (IR drop, sneak path, ADC 양자화)
- **시간적 비이상성 미반영** (drift, endurance)
- **VTEAM 동역학 미적분** (정적 mapping에만 활용. 제목도 그래서 "VTEAM 기반"이 아닌 "4종 멤리스터 소재 파라미터 활용"으로 변경됨)
- **표본 한계** (3 baseline × 3 noise = 9회/소재. 통계 유의성 검정 미수행)
- **Variation 단순화** (LRS/HRS σ 산술 평균)
- **NW 5% fault rate**가 다른 소재(0.3~0.5%)보다 10배 큼 (다만 NW는 ideal에서도 망하므로 본질 한계가 양자화임을 본문에서 논증)

## 6. 외부 검토 8라운드 합의 기록 (results/ 폴더 참고)

라운드별 prompt와 결과:
- v1→v2 (Phase 2): 친구 수준으로 단순화
- v2→v3 (옵션 D): 15개 신규 지적(N1~N15) 100% 반영
- v3→v4 (옵션 P3): 15개 신규 지적(M1~M15) 100% 반영
- v4→v5: 잔존 3건 반영 (HTML title, Table 4 caption 등)
- v5→v6 (Q1~Q8 + R1~R4): Subset Acc 추가, 표현 톤다운
- v6→v7: per-class F1 Table 6 추가

이 모든 토론 prompt는 `results/codex_*.md`, `results/gemini_*.md`로 저장되어 있음.

### 3자 합의된 핵심 결정사항
- **제목에서 VTEAM 제거** (실제 동역학 미사용. 정직성)
- **Table 4에 Subset Acc 추가** (cherry-picking 의심 해소)
- **NW 5% fault = sensitivity bound로 framing**
- **"통계적 신뢰성 확보" 표현 톤다운** (3 seed로는 부족)
- **"근본적 우회" → "완화"** (PIM 과장 톤다운)

## 8. 만약 이어서 할 때 가능한 작업 (사용자가 결정 시)

### 단기 (각 30분~1시간)
- 발표 슬라이드 8~10장 (디펜스용)
- paper.html → PDF 변환 (제출 형식)
- README 영문 보강 (포트폴리오 노출용)
- requirements.txt 추가 (재현성)
- LICENSE 파일 (MIT, GitHub Public 표준)

### 중기 (학회 투고 고려 시, 각 1~3일)
- 저항비 합성 sweep 실험 (5×, 10×, 20×, 50×, 100×, 300×)
- Monte Carlo 30~50회 확대 + CVaR/5th percentile
- HRS/LRS variation 비대칭 분리 모델
- Noise-Aware Training 실제 구현 + 효과 검증
- NW fault=0.5% 통일 통제 실험 추가
- WM-811K 데이터셋에서 동일 파이프라인 재실험

### 장기 (학회 논문 1편 분량)
- 회로 수준 비이상성 (IR drop, sneak path) 추가
- QAT vs PTQ 비교
- Triple redundancy / row-level ECC 효과 정량화
